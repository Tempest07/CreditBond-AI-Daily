# 这是真 AI，不是规则策略

这个项目里有两层东西：

## 1. 神经网络预测器

`creditbond_ai/model.py` 里的 `LSTMClassifier` 是真实神经网络：

- LSTM 层学习时间序列依赖。
- LayerNorm、Dropout、Linear head 负责稳定训练和输出三分类 logits。
- 训练时用交叉熵损失、AdamW 优化器、反向传播更新权重。
- 预测时输出 `看空 / 看多 / 震荡` 的概率分布。

这部分不是规则，不是阈值打分，也不是传统技术指标信号。

## 2. 金融实验框架

监督学习必须先定义训练标签，所以我们仍然需要金融上的目标定义：

- 未来 X 天收益率上行超过阈值：看空。
- 未来 X 天收益率下行超过阈值：看多。
- 未来 X 天收益率变化较小：震荡。

回测 proxy 也只是评价工具，用来检查模型输出有没有方向价值。它不参与模型预测，不会把手写策略塞进神经网络里。

## 下一步可升级

第一版用 LSTM 是因为它稳、数据需求相对低、适合先跑通数据闭环。现在主线已经改为 DM API 全接管，等 DM 数据覆盖稳定后，可以加入：

- GRU
- Transformer Encoder
- Temporal Fusion Transformer
- 多任务模型，同时预测 1 天、5 天、20 天方向
- 回归 + 分类混合模型，同时预测收益率变化幅度和方向
